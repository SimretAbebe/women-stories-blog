# 🌟 Women's Journeys Blog

A premium platform dedicated to sharing and celebrating the powerful stories of women's journeys. Built with a focus on **visual excellence**, **smooth performance**, and **inclusive storytelling**.

---

## 🛠 Tech Stack

**Frontend:**
- [React.js]
- [Tailwind CSS]
- [Lucide Icons]
- [Axios]

**Backend:**
- [Django]
- [Django REST Framework]
- [Simple JWT]
- [SQLite / PostgreSQL]
